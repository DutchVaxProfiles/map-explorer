# Preview Data

The current public map files are for the conference demo only.

- `buurt_2026.geojson` is temporary preview geometry copied from the earlier
  vaxprofiles map fork's `buurt_2024.geojson`.
- `buurt_5_processed.csv` contains deterministic synthetic values for five
  profile groups.
- `preprocess_report.json` records the placeholder status and row counts.

Replace these files by running `scripts/preprocess_map_data.py` once the real
CBS/export CSV and current buurt GeoJSON are available.
